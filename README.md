## Introduction 
# Unified-Android-Tweaker
A universal and unified Android tweakers module. Re-configured, modified into one convenient package. 

Feature:
- Fine-tune system kernel tunable parameters (Configurable).
- Modify Thermal System (replace/update files and configs by OverlayFS).
- Modify Audio System to allow any 3rd-party Audio Mods (EQFX, Viper4Android, etc) process as system-wide audio.
- BootLoop Protection function (Hardware Key Buttons).
- Systemless.
- Open source.
